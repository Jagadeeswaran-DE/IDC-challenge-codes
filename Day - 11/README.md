### 2. Hypothesis Testing (Weekday vs Weekend)

A simple business question was tested:

> Do users behave differently on weekdays compared to weekends?

Steps followed:

1. Categorized data into weekday and weekend
2. Counted user actions (views, carts, purchases)
3. Compared results instead of relying on assumptions

This demonstrated how **data can validate or reject beliefs**.

### 3. A/B Testing Concepts

* Control group (A)
* Test group (B)
* Measuring impact using metrics

A/B testing helps organizations make **safe, data-driven changes**.
